# hexNoteG9
This is a format for music notation in hex.

|	HEX	|	Key for HexNote Values 	|
| ------------- | ------------- |
|	0	|	Comment/Goto 	|
|	1	|	Sharp/Flat	|
|	2	|	Dynamics 	|
|	3	|	Accents 	|
|	4	|	Defaults	|
|	5	|	Parens	|
|	6	|	Key Change/Chord	|
|	7	|	Octave	|
|	8	|	Rest	|
|	9	|	Note G	|
|	a	|	Note A	|
|	b	|	Note B	|
|	c	|	Note C	|
|	d	|	Note D	|
|	e	|	Note E	|
|	f	|	Note F	|
